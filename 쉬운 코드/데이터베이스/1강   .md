- 데이터베이스란
	- 전자적으로 저장되고 사용되는 관련있는 데이터들의 조직화된 집합
		- 관련있는 데이터란 같은 출처나 목적, 서비스안에서 생성되는 데이터들
		- 조직화된 집합이란 데이터를 찾기 쉽도록 조직화
		- 데이터가 컴퓨터에 저장되므로 전자적
- DBMS란
	- Database Management System
		- DB를 정의, 생성, 관리하는 기능 제공
		- DB를 정의하다 보면 부가적인 데이터인 메타데이터 발생
			- 메타데이터란
				- 데이터를 설명하기 위한 데이터
				- db를 정의하거나 기술하는 데이터
				- 카탈로그라고도 부름
				- 메타데이터 또한 DBMS로 관리
