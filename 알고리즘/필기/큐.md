- 성질
	- 한쪽 끝에서 원소를 넣고, 반대쪽 끝에서 원소를 뺄 수 있는 자료구조
	- 먼저 들어간 원소가 먼저 나옴 (FIFO)
- 연산
	- O(1)
		- 원소의 추가
		- 원소의 제거
		- 제일 앞/뒤의 원소 확인
	- 제일 앞/뒤가 아닌 나머지 원소들의 확인/변경이 원칙적으로 불가능
- 구현
  ```c++
  const int MX = 100005;
  ind dat[MX];
  int head = 0, tail = 0;
```
	- 원소를 담을 큰 배열 하나
	- 원소의 앞쪽과 뒤쪽을 가리킬 변수 두개
	- 큐의 크기 =
