- 성질
	- k번째 원소 확인 / 변경 위해 O(k)가 필요
	- 임의의 위치에 원소 추가, 제거는 O(1)
	  단, 추가하고 싶은 원소의 주소를 알고 있을 때
	- 원소들이 메모리 상에 연속해있지 않아 [[Cache Hit Rate]]가 낮지만 할당이 다소 쉬움
	- 텍스트 에디터와 같이 임의의 위치에 원소를 추가, 제거가 활발한 경우에 사용
